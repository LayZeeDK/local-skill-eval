| Property | Value |
|----------|-------|
| Framework | ts-node with manual test scripts (project convention) |

## Sampling Rate
- **Per experiment:** 1 warm-up + 3 measured trials
- **Per wave completion:** Full 3-trial run with winning config
- **Phase gate:** 3 consecutive trials averaging < 300s with reward >= 0.90
